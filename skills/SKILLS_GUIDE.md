# 🚀 The Complete AI Skills Encyclopedia

Ye guide aapke pas mojud tamam 36+ AI powers ka mukammal nichore hai.

---

## 🏗️ Phase 1: Bootstrap & Architecture (The Foundation)
*Naya project shuru karne ke liye.*

*   **React Native Clean Bootstrap:** Mobile app setup (CLI, No Expo, Strict Types).
*   **React Web Clean Bootstrap:** Web app setup (Vite, Tailwind, Modular Routes).
*   **Backend Clean Bootstrap:** Node/Prisma & Python setup (MVC Pattern).
*   **V3 DDD Architecture:** Domain-Driven Design follow karne ke liye.
*   **V3 Core Implementation:** System ki buniyadi core files setup karne ke liye.

---

## 🤖 Phase 2: Workflow & Swarm Orchestration (The Engine)
*Aik team ki tarah kaam karne ke liye.*

*   **Get An Issue And Work On:** Trello se khud hi ticket utha kar code likhna.
*   **Triage Module Tickets:** Trello cards ko prioritize karna.
*   **Swarm Orchestration & Advanced Swarm:** 8-10 agents ko mil kar aik bare task par lagana.
*   **Flow Nexus (Platform, Neural, Swarm):** AI agents ka aik "Neural Network" banana jo complex projects ko manage kare.
*   **Hooks Automation:** Kaam se pehle aur baad mein auto-scripts chalana.

---

## 🛡️ Phase 3: Quality, Security & Verification (The Guard)
*Code ko safe aur bugs-free rakhne ke liye.*

*   **Healthcare Platform QA:** Full UI + security audit — har role (Customer/Staff/Admin) ka browser test + backend IDOR/auth/injection checks. `/healthcare-qa`
*   **Verification & Quality Assurance:** Truth scoring (accuracy check) aur auto-rollback.
*   **GitHub Code Review:** PRs ka senior-level review karna.
*   **GitHub Workflow Automation:** CI/CD pipelines aur GitHub actions manage karna.
*   **V3 Security Overhaul:** Code mein security vulnerabilities check kar ke theek karna.
*   **ReasoningBank Intelligence:** AI ki logic aur reasoning ko check karna.

---

## 🧠 Phase 4: Intelligence, Memory & Learning (The Brain)
*AI ko smart aur hamesha seekhne wala banane ke liye.*

*   **AgentDB (Advanced, Learning, Memory, Optimization, Vector Search):** AI ki yaaddasht (Memory) manage karna taake wo purane kaamo se seekhay.
*   **V3 Memory Unification:** Tamam agents ki memory ko aik jagah jama karna.
*   **ReasoningBank AgentDB:** Knowledge ko fast search karne ke liye vector database use karna.
*   **Skill Builder:** AI ko nai skills khud sikhane ke liye.

---

## 🛠️ Phase 5: Specialized Tools & Integrations (The Specialist)
*Khaas kism ke kaamo ke liye.*

*   **Browser Skill:** Live website testing aur screenshots.
*   **Stripe (Best Practices, Projects, Upgrade):** Payments aur billing system handle karna.
*   **GitHub (Release, Project Management):** Version control aur releases manage karna.
*   **V3 Performance Optimization:** Code ki speed barhane ke liye (Flash Attention, etc.).
*   **EOD Report:** Din ke ikhtitam par poori report generate karna ke kya kaam hua.

---

## 🤝 Phase 6: Collaborative & Methodology
*Human-AI collaboration.*

*   **Pair Programming:** User aur AI ka mil kar live debugging karna.
*   **SPARC Methodology:** Baray tasks ko Spec, Plan, Action, Review mein baant kar karna.
*   **Stream Chain:** Kaam ko aik silsilay (chain) mein chalana.

---

## 💡 How to use this Encyclopedia?
Aap mujhse kisi bhi waqt keh sakte hain:
> *"Jani, `V3 Performance Optimization` lagao is backend par aur check karo `AgentDB` mein isse milta julta koi purana code hai ya nahi?"*

Main in sab ko mil kar use karunga! 🚀🦾⚡️
